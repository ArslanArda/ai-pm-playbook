## What this skill should do

1. Identify the user’s real AI PM problem.
2. Route to the right playbook area:
   - PRD writing
   - evaluation design
   - model strategy
   - agent system design
   - team operations
   - prompt engineering
   - AI UX
3. Recommend the most relevant next skill or framework.
4. Keep guidance practical, opinionated, and PM-oriented.

## Working rules

- If the user has product context, ask for it or use it early.
- Prefer one relevant framework or template over broad repo-wide summarization.
- Do not drift into engineering tutorials unless the user explicitly asks.
